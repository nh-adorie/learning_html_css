Học github thật dễ ^_^

git init: tạo repo ở máy

git status: kiểm tra xem có file nào

git add index.html để thêm file đấy vào

git commit -m"Ghi nội dung chú thích ở đây"

git remote add origin https://github.com/nh-adorie/gihub-guide.git

Cái link đấy lấy ở trên website github

git push -u origin main
